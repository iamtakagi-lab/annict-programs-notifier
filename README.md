# annict-programs-notifier
その日に放送されるアニメの番組情報を Annict から通知します。

`見てる` のステータスが選択されている作品の放送予定を通知します。[チャンネル一覧 | Annict](https://annict.com/channels) にて適宣設定の上で使用してください。

Cron で定期的に呼び出してください。

## Build & Run with Docker
`docker-compose up -d --build`
